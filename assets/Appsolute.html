ERP on Cloud รายแรกของไทย

เกี่ยวกับบริษัท

แอ๊บโซลูท กำเนิดขึ้นในปี 2547 ยืนอยู่บนความเชื่อที่ว่า Cloud จะเป็นเทคโนโลยีที่เปิดโอกาสให้ SMEs สามารถมีระบบบริหารจัดการธุรกิจ (ERP) ที่ดีได้

จึงเป็นจุดกำเนิดของระบบ ABSOLUTE SOLUTIONS (|SOLn|) ที่ถือเป็นระบบบริหารจัดการธุรกิจออนไลน์บน Cloud รายแรกของไทย

รางวัลที่ได้รับ

รางวัลชนะเลิศ จากโครงการ "True Cloud Winner Project" ได้รับการรับรองจาก SOFTWARE PARK (THAILAND) ว่าเป็น ERP on Cloud รายแรกของไทย
